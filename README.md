# 🪙 Gold Price Tracker (CoinGecko API)

A simple live *Gold Price Tracker* using CoinGecko API.  
Shows *24K & 22K gold prices per gram* in INR and USD.

---

## ⚙ Features
- Live gold price (24K & 22K)
- INR and USD support
- Real-time data via CoinGecko API
